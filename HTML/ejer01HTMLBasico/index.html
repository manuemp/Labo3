Preguntas:

1- Ejemplo de uso de etiquetas Meta

Las etiquetas Meta se utilizan para metadatos, que son datos relacionados al documento pero que no son visualizados en la presentación del mismo.
En estas etiquetas se puede indicar el formato de codificación de los caracteres de la página, el comportamiento de la página según el tamaño del dispositivo, la descripción de la página o su categoría para un mejor posicionamiento, entre otras.

2- Atributos aplicables a la etiqueta HTML

  • lang: Define el idioma principal del documento
  • dir: Especifica la dirección del texto
  • title: Brinda información adicional al usuario cuando deja mantenido el cursor sobre la página
  • Atributos aplicables al resto de las etiquetas, como style, id, class

3- ¿Cómo se relaciona un header HTTP con la etiqueta Head de HTML?

Debido a que no se pueden enviar headers HTTP desde un HTML, en una etiqueta meta se escribe un dato que le dice al navegador los headers HTTP que se van a utilizar.
Http-equiv (equiv = equivalente) es el atributo que me premite hacer referencia a un header HTTP; estoy dando desde HTML un dato que normalmente va en el HTTP.

4- ¿Qué es UTF8?

UTF 8 es un formato de codificación de caracteres que tiene la capacidad de representar cualquier caracter Unicode. En el caso de páginas en español latino, este formato de codificación permite el uso de tildes y la letra ñ sin la aparición de caracteres no deseados en el documento.

5- ¿Qué atributo HTML sirve para aplicar estilo y a qué etiqueta es aplicable?

El atributo style sirve para aplicar estilo al documento (color, tamaños de fuente, tamaño de imágenes, etc.), y es aplicable a todas las etiquetas que contengan elementos o datos que podrán ser visualizados de alguna forma en el documento.

6- ¿Dónde se indica que un enlace debe ubicar el documento en una nueva pestaña?

Con el valor "_blank" en el atributo target de la etiqueta "a" se indica que un link deberá ser abierto en una pestaña nueva.

7- Explicar cuándo se usarían etiquetas semánticas y con qué propósito

Las etiquetas semánticas se utilizan para la optimización del sitio para los motores de búsqueda. Sirven para enmarcar o estructurar semánticamente el contenido del html dentro del body. La utilización de las etiquetas semánticas generalmente responde a formas estandarizadas de presentar el contenido de una página, como por ejemplo la etiqueta footer suele contener información como direcciones, números de documento, contacto, la etiqueta main indica dónde está la información más relevante del documento, la etiqueta section permite dividir en secciones los distintos tipos de contenido del documento, etc. El contenido que haya dentro de cada etiqueta semántica permite que una página tenga una posición más relevante a la hora de filtrar contenido específico en un buscador.

8- ¿Qué atributos de estilo básicos aplicarían a una tabla? Explicar

Los atributos de estilo básicos aplicables a una tabla son:

  • border: Modifico el tipo de borde de una tabla (sólidos, punteados, de guiones, etc.)
  • border-collapse: Los bordes de las celdas "colapsan", se combinan en uno solo
  • width: Indica el ancho de la tabla
  • height: Indica el alto de la tabla
  • background-color: Indica el color de fondo de la tabla

9- ¿Qué diferencia existe entre la etiqueta body de un documento HTML y el body del protocolo HTTP?

La etiqueta body se encuentra dentro del HTML y engloba al contenido principal donde se encontrarán el resto de las etiquetas que harán a la forma y disposición del documento HTML. En cambio, el body del protocolo HTTP contiene datos relacionados a peticiones de HTTP, que no tienen relación con el lenguaje HTML. Dentro del body de una respuesta HTTP se puede encontrar el archivo HTML donde está el body con el contenido del documento, pero más allá de eso, nada tienen que ver uno con el otro.

10- ¿Qué atributos marcan la agrupación de celdas de una tabla? ¿A qué etiquetas se aplican?

El atributo colspan me permite indicar cuántas celdas se van a querer agrupar horizontalmente, y el atributo rowspan indica cuántas celdas se van a agrupar verticalmente. Ambas se aplican a las etiquetas td.
